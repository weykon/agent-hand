Agent Hand

Agent Hand は、複数のAIコーディングエージェントセッション（Claude Code、Gemini CLI、Cursor、Codex、Windsurf）を1つのtmuxベースのダッシュボードで管理するオープンソースのRust TUIです。起動時間50ms未満。

優先ジャンプ、ファジー検索、キャンバスワークフローエディタ、AIセッションサマライザー、マルチツールフック。コア機能は無料、ProとMaxティアあり。

curl -fsSL https://raw.githubusercontent.com/weykon/agent-hand/master/install.sh | bash
agent-hand

●research-agent[コーディング]思考中...
✓bugfix-session[デバッグ]5分前に完了
○new-project[実験]未開始

よくある質問

Agent Handとは？

Agent Handは、Claude Code、Gemini CLI、Cursor、Codex、WindsurfなどのAIコーディングエージェント向けの高速なRustベースのターミナルセッションマネージャー（TUI）です。tmuxで分離し、ステータス検出、優先ジャンプ、ファジー検索、キャンバスワークフローエディタ、AIセッションサマライザーを備えた1つのダッシュボードから5つ以上の同時AIセッションを管理できます。

Agent Handをインストールするには？

次のコマンドを実行：curl -fsSL https://raw.githubusercontent.com/weykon/agent-hand/master/install.sh | bash — macOS、Linux、WSLで動作します。または cargo build --release でソースからビルドできます。

Agent Handは無料ですか？

はい、コアのセッションマネージャーはオープンソース（MIT）で無料です。ステータス検出、優先ジャンプ、ファジー検索、マルチツールフック、中国語サポートが含まれます。Pro（$19買い切り）でキャンバスワークフローエディタ、セッション関係、スキルライブラリ、サウンド通知、セッション再開、リモート共有が追加されます。Max（$9/月）でAIセッションサマライザー、コラボレーティブコンテキストキャプチャ、リモートセッションノードが追加されます。

Agent HandはClaude Codeで使えますか？

はい。Agent Handは元々複数のClaude Codeセッションを管理するために作られました。セッションステータス（待機中、実行中、アイドル）を検出し、Ctrl+Nで入力が必要なセッションに優先ジャンプし、Ctrl+Gですべてのセッションをファジー検索できます。

Agent Handとtmuxの違いは？

Agent Handはtmuxの上で動作します（設定に影響しないよう専用サーバーを使用）。AIエージェント固有の機能を追加します：自動ステータス検出、優先ジャンプ、セッショングルーピング、PTYモニタリング、ビジュアルダッシュボード — これらは素のtmuxにはない機能です。

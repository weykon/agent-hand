対応ツール

Agent Handと連携するAIコーディングエージェントおよび開発ツール。

AIコーディングエージェント

Agent Handはtmuxセッションを作成するあらゆるツールと連携します。以下はフック統合がファーストクラスでサポートされているエージェントです：

Claude Code（Anthropic）

Agent Handはフックシステムを通じてClaude Codeとの深い統合を提供します：

  • SessionStartでのセッションID自動キャプチャ
  • プロンプト送信イベントの追跡
  • セッションの停止/再開ライフサイクルイベント

Cursor

Cursorはコマンド実行用にターミナルセッションを作成します。Agent Handはtmuxを通じてこれらを自動的に検出します。

GitHub Copilot / Codex

tmuxセッション検出により動作します。CopilotまたはCodexが生成したターミナルセッションは自動的にリストに表示されます。

Windsurf（Codeium）

Windsurfのターミナルセッションも同じ方法でキャプチャされます。Agent Handはtmuxサーバー上のすべてのセッションを監視します。

OpenCode

tmuxセッション検出により対応しています。

検出の仕組み

Agent Handは2つの補完的なメカニズムを使用しています：

  1. tmuxポーリング — 設定されたtmuxサーバー上のすべてのセッションを定期的にリストし、状態を検出します
  2. フックイベント — 対応ツール（現在はClaude Code）では、イベントフックがセッションID、タイトル、ライフサイクルイベントなどのリッチなメタデータを提供します

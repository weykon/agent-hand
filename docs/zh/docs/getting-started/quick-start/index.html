入门指南

快速开始

启动 Agent Hand 并在 2 分钟内学会基本操作。

启动

TUI 启动后，左侧是会话列表，底部是信息栏。

核心快捷键

按键操作
j / k在会话列表中上下导航
Enter连接到选中的会话（跳入 tmux）
q退出 Agent Hand
/模糊搜索会话
r刷新会话列表
g分组管理器
?显示所有快捷键

会话状态

每个会话都会显示彩色的状态指示器：

  • 运行中（绿色）— 进程正在执行
  • 空闲（蓝色）— 会话存在但没有进程在运行
  • 已退出（灰色）— 进程已完成
  • 错误（红色）— 进程以非零退出码退出

Agent Hand 通过检查 tmux 窗格自动检测这些状态。

优先级排序

会话按优先级排序：错误和正在运行的会话排在最前面，因此最重要的内容始终在顶部。

配合 AI 代理使用

当你运行 AI 编程代理（Claude Code、Cursor 等）时，它通常会创建多个终端会话。Agent Hand 会自动发现并列出所有 tmux 会话，包括 AI 工具创建的会话。

典型工作流程：

  1. 在一个终端中启动你的 AI 代理
  2. 在另一个终端中打开 Agent Hand（或使用 tmux 分屏）
  3. 观察会话出现在列表中
  4. 跳入任意会话查看输出
  5. 利用状态指示器快速发现错误

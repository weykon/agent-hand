Agent Hand

Agent Hand is an open-source Rust TUI that manages multiple AI coding agent sessions — Claude Code, Gemini CLI, OpenCode — from one tmux-backed dashboard with <50ms startup.

Priority jump to urgent sessions, fuzzy search across all agents, real-time status detection, and PTY monitoring. Free and MIT-licensed.

Frequently Asked Questions

What is Agent Hand?

Agent Hand is a fast, Rust-based terminal session manager (TUI) for AI coding agents like Claude Code, Gemini CLI, and OpenCode. It uses tmux for isolation, letting you manage 5+ concurrent AI sessions from one dashboard with status detection, priority jump, and fuzzy search.

How do I install Agent Hand?

Run: curl -fsSL https://raw.githubusercontent.com/weykon/agent-hand/master/install.sh | bash — works on macOS, Linux, and WSL. Or build from source with cargo build --release.

Is Agent Hand free?

Yes, the core session manager is free and open source (MIT). There is an optional Pro tier ($19 one-time) for power features and a Max tier ($9/month) for AI summarizer, remote sharing, and session relationships.

Does Agent Hand work with Claude Code?

Yes. Agent Hand was originally built to manage multiple Claude Code sessions. It detects session status (Waiting, Running, Idle), lets you priority-jump to sessions needing input with Ctrl+N, and provides fuzzy search across all sessions with Ctrl+G.

How is Agent Hand different from tmux?

Agent Hand runs on top of tmux (using a dedicated server so it won't touch your config). It adds AI-agent-specific features: automatic status detection, priority jumping, session grouping, PTY monitoring, and a visual dashboard — things raw tmux doesn't provide.

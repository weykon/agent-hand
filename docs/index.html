Get Started

One-Liner Install

macOS / Linux / WSL

curl -fsSL https://raw.githubusercontent.com/weykon/agent-hand/master/install.sh | bash

Build from Source

Rust toolchain required

git clone https://github.com/weykon/agent-hand.git
cd agent-hand
cargo build --release

Requirements

  • tmux - The install script will attempt to install it automatically

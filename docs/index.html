Key Features

🔍

Status Detection

Clear visual indicators show WAITING, RUNNING, READY, or IDLE status at a glance.

! ● ✓ ○
🎯

Priority Jump (Ctrl+N)

Instantly jump to the most urgent session. Never miss a confirmation prompt again.

⚡

Fuzzy Switcher (Ctrl+G)

Search and switch to any session in milliseconds. Type a few chars, jump directly.

📊

PTY Monitoring

Real-time pseudo-terminal usage tracking with system-wide gauge and warnings.

📁

Groups & Labels

Organize sessions by project with custom titles and colored labels.

🔒

tmux Isolation

Dedicated tmux server won't touch your default tmux. Your configs, your workflow.

Get Started

One-Liner Install

macOS / Linux / WSL

curl -fsSL https://raw.githubusercontent.com/weykon/agent-hand/master/install.sh | bash

Build from Source

Rust toolchain required

git clone https://github.com/weykon/agent-hand.git
cd agent-hand
cargo build --release

Requirements

  • tmux - The install script will attempt to install it automatically

From the Creator

"In early 2025, I was juggling 5+ Claude Code instances simultaneously. Four terminal windows, each with 3-4 tmux panes. 'Did I already respond to that prompt?' 'Which Claude is working on which task?' Wasting 10+ minutes just finding the right session."

"I tried the original agent-deck (Go) and loved the concept, but wanted better performance, more features, and cleaner integration. Agent Hand was born — a Rust rewrite that keeps what works and adds what I needed."

"The best tool is the one you'll actually use."

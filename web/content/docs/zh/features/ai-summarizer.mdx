## 概述

AI 摘要功能会分析你终端会话的内容，并生成简洁的活动摘要。当你有许多会话在运行，需要快速了解每个会话的状态时，这非常有用。

## 工作原理

1. 在列表中选择一个会话
2. 按 `A` 打开 AI 分析选择器
3. 选择**摘要**获取文本摘要，或选择 **ASCII 图表**获取可视化表示
4. AI 处理会话的终端输出
5. 结果显示在弹出窗口中

## 分析模式

### 摘要

会话活动的简洁文本摘要：

- 执行了哪些命令
- 关键输出和结果
- 遇到的错误或警告
- 整体状态评估

### ASCII 图表

会话活动流程的可视化表示，以 ASCII 图表的形式直接在终端中渲染。

## AI 提供者

Agent Hand 使用可配置的 AI 提供者进行摘要生成。系统支持：

- 多种提供者后端
- 自定义 API 端点
- Token 感知的上下文管理

在设置对话框中配置你的 AI 提供者（按 `S`）。

## 隐私

会话内容会发送到配置的 AI 提供者进行分析。Agent Hand 的服务器不会存储任何数据。你可以控制由哪个提供者处理你的数据。

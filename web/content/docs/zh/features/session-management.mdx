## 会话列表

主面板显示所有已发现的 tmux 会话，包含：

- **会话名称** — tmux 会话标识符
- **状态** — 运行中、空闲、已退出或错误（彩色标识）
- **最后活动时间** — 会话上次活跃的时间
- **分组标记** — 如果会话属于某个分组

## 模糊搜索

按 `/` 打开模糊搜索。输入关键词即可按名称、分组或标签过滤会话。搜索是即时的，可覆盖所有会话元数据。

## 分组

分组让你可以逻辑地组织会话。按 `g` 打开分组管理器。

### 创建分组

1. 按 `g` 打开分组
2. 按 `n` 创建新分组
3. 输入分组名称
4. 选择要添加的会话

### 自动分组

Agent Hand 可以根据命名模式自动对会话进行分组。来自同一个 AI 代理或项目的会话会被自动归为一组。

## 标签

标签是你可以附加到会话上的轻量级标记，用于快速识别。与分组不同，一个会话可以有多个标签。

## 优先级排序

会话按优先级自动排序：

1. **错误** — 非零退出码的会话（显示在最前）
2. **运行中** — 正在执行的进程
3. **空闲** — 等待输入
4. **已退出** — 已完成的进程（显示在最后）

这确保你始终能最先看到最重要的会话。

## 会话操作

| 按键 | 操作 |
| --- | --- |
| `Enter` | 连接到会话 |
| `d` | 从会话断开 |
| `x` | 终止会话（需确认） |
| `g` | 管理分组 |
| `/` | 模糊搜索 |
| `r` | 刷新列表 |
| `s` | 排序选项 |

## 多工具支持

当你同时使用多个 AI 代理时，Agent Hand 将它们的所有会话整合到一个视图中。每个代理的会话通过其命名模式和可选的分组分配来区分。

## 概述

画布是一个内置于终端的可视化编辑器。它让你可以创建工作流图、会话地图和节点图——全部以 ASCII/Unicode 字符画的形式呈现。

按 `p` 切换画布面板。

## 画布节点

节点代表工作单元。每个节点包含：

- **标题** — 简短描述
- **内容** — 详细文本（支持多行）
- **位置** — 画布上的 x/y 坐标
- **连接** — 与其他节点的边

### 创建节点

1. 按 `p` 打开画布
2. 按 `n` 创建新节点
3. 输入节点标题
4. 使用方向键调整位置
5. 按 `Enter` 放置节点

### 编辑节点

选择一个节点并按 `e` 编辑其内容。编辑器支持多行文本和自动换行。

## 边（连接线）

通过边连接节点以显示关系和流程。

1. 选择源节点
2. 按 `c` 开始连接
3. 导航到目标节点
4. 按 `Enter` 创建边

### 边标签

边可以添加标签。在选中的边上按 `l` 添加或编辑标签。标签显示在边的路径上。

## 会话地图

画布可以自动生成**会话地图**——你活跃会话及其关系的可视化表示。它基于以下数据构建：

- 会话生命周期事件（启动、停止、恢复）
- 会话之间的关系数据
- AI 生成的摘要

## 视口导航

| 按键 | 操作 |
| --- | --- |
| 方向键 | 平移视口 |
| `+` / `-` | 放大/缩小（即将推出） |
| `h` | 重置视口到初始位置 |
| `f` | 将所有节点适配到视图中 |

## AI 集成（Max）

使用 Max 层级，你可以：

- 使用 AI 生成节点摘要
- 基于会话关系自动布局节点
- 以类型着色样式可视化关系边

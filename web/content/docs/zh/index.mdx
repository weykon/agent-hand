## 什么是 Agent Hand？

Agent Hand 是一个用 Rust 编写的终端会话管理器，它为你提供统一的 TUI（终端用户界面）来管理由 AI 编程代理（如 **Claude Code**、**Cursor**、**Codex**、**Windsurf** 等）创建的会话。

当你使用 AI 驱动的开发工具时，它们会启动用于构建、测试、开发服务器和其他任务的终端会话。Agent Hand 将所有这些会话整合到一个仪表板中，让你可以：

- 查看每个会话的**实时状态**（运行中、空闲、已退出、错误）
- **一键跳转到任意会话**
- 使用**分组和标签**组织会话
- 获取会话的 **AI 生成摘要**
- 从会话活动中构建**画布工作流图**
- 通过中继服务**跨设备共享会话**

## 架构

Agent Hand 构建在 **tmux** 之上——它不会重新发明终端复用器，而是在其上添加了一个智能管理层：

```
┌─────────────────────────────────────┐
│         Agent Hand TUI              │
│  ┌─────┐ ┌─────┐ ┌─────┐ ┌──────┐ │
│  │列表 │ │信息 │ │画布 │ │搜索/ │ │
│  │面板 │ │栏   │ │编辑 │ │过滤  │ │
│  └──┬──┘ └──┬──┘ └──┬──┘ └──┬───┘ │
│     │       │       │       │      │
│  ┌──┴───────┴───────┴───────┴──┐   │
│  │      tmux 服务器             │   │
│  │  会话1    会话2    ...      │   │
│  └─────────────────────────────┘   │
└─────────────────────────────────────┘
```

## 功能层级

Agent Hand 采用 **Free / Pro / Max** 三级模型：

| 层级 | 包含内容 |
| --- | --- |
| **Free** | 会话管理、状态检测、分组、钩子、国际化、模糊搜索 |
| **Pro** | 画布工作流编辑器、会话关联、技能、声音通知、会话恢复 |
| **Max** | AI 会话摘要、协作上下文、远程会话节点 |

## 快速开始

```bash
# 安装（macOS / Linux）
curl -fsSL https://raw.githubusercontent.com/anthropics/agent-hand/main/install.sh | bash

# 启动
agent-hand
```

查看[安装指南](/docs/getting-started/installation)了解各平台的安装说明，以及[快速开始](/docs/getting-started/quick-start)获取 TUI 的使用教程。

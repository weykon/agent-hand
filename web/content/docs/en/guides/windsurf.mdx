## Overview

Windsurf by Codeium is an AI-powered IDE that runs terminal commands for code execution and testing. Agent Hand discovers these sessions automatically through tmux.

## Setup

1. Install Agent Hand ([Installation](/docs/getting-started/installation))
2. Ensure Windsurf's terminal sessions run in tmux
3. Launch Agent Hand: `agent-hand`

## Workflow

1. Open your project in Windsurf
2. Run Agent Hand in a separate terminal
3. Windsurf's terminal sessions appear automatically
4. Monitor, jump to, and manage sessions from Agent Hand

## Tips

- **Status at a glance**: See which Windsurf tasks are running, completed, or failed
- **Error triage**: Errors bubble to the top of the list — press `Enter` to investigate
- **Multi-tool setup**: Run Windsurf alongside other AI tools and see all sessions in one place

## Agent Handとは？

Agent Handは、Rustで書かれたターミナルセッションマネージャーです。**Claude Code**、**Cursor**、**Codex**、**Windsurf**などのAIコーディングエージェントが生成したセッションを、統一されたTUI（ターミナルユーザーインターフェース）で管理できます。

AI搭載の開発ツールを使用すると、ビルド、テスト、開発サーバーなどのタスクのためにターミナルセッションが起動されます。Agent Handはこれらすべてを1つのダッシュボードにまとめ、以下のことが可能になります：

- すべてのセッションの**ライブステータス**を確認（実行中、アイドル、終了、エラー）
- キー1つで**任意のセッションにジャンプ**
- **グループとラベル**でセッションを整理
- セッションの内容を**AIが自動要約**
- セッションの活動から**キャンバスワークフロー図**を作成
- リレーを通じてマシン間で**セッションを共有**

## アーキテクチャ

Agent Handは**tmux**の上に構築されています。ターミナル多重化を再発明するのではなく、その上にスマートな管理レイヤーを追加しています：

```
┌─────────────────────────────────────┐
│         Agent Hand TUI              │
│  ┌─────┐ ┌─────┐ ┌─────┐ ┌──────┐ │
│  │List │ │Info │ │Canv.│ │Search│ │
│  │Panel│ │Bar  │ │Edit.│ │/Filt.│ │
│  └──┬──┘ └──┬──┘ └──┬──┘ └──┬───┘ │
│     │       │       │       │      │
│  ┌──┴───────┴───────┴───────┴──┐   │
│  │      tmux server(s)         │   │
│  │  session1  session2  ...    │   │
│  └─────────────────────────────┘   │
└─────────────────────────────────────┘
```

## 機能ティア

Agent Handは **Free / Pro / Max** のティアモデルを採用しています：

| ティア | 利用可能な機能 |
| --- | --- |
| **Free** | セッション管理、ステータス検出、グループ、フック、i18n、ファジー検索 |
| **Pro** | キャンバスワークフローエディタ、セッション関係、スキル、サウンド通知、レジューム |
| **Max** | AIセッション要約、コラボレーティブコンテキスト、リモートセッションノード |

## クイックスタート

```bash
# インストール（macOS / Linux）
curl -fsSL https://raw.githubusercontent.com/anthropics/agent-hand/main/install.sh | bash

# 起動
agent-hand
```

プラットフォーム別の手順は[インストール](/docs/getting-started/installation)を、TUIのウォークスルーは[クイックスタート](/docs/getting-started/quick-start)をご覧ください。

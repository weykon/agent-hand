## 概要

CursorはAI搭載のコードエディタで、コード生成、デバッグ、テストのためにターミナルコマンドを実行します。これらのターミナルセッションはtmux内で実行され、Agent Handが自動的に検出します。

## セットアップ

1. Agent Handをインストール（[インストール](/docs/getting-started/installation)）
2. Cursorがターミナルセッションにtmuxを使用していることを確認
3. Agent Handを起動: `agent-hand`

Agent HandはCursorのターミナルセッションを自動的に検出します。

## ワークフロー

1. Cursorでプロジェクトを開く
2. 別のターミナルでAgent Handを起動
3. Cursorがコマンド（ビルド、テスト、リンター）を実行すると、Agent Handに表示される
4. Agent Handでステータスの監視、エラーへのジャンプ、出力の確認が可能

## ヒント

- **ビルドステータスの監視**: 赤（エラー）インジケーターでビルドの失敗を検知
- **クイック確認**: セッションで `Enter` を押すとターミナル出力の全体を表示
- **グループで整理**: 「Cursor」グループを作成して、Cursorセッションを他のツールと分離

## 概要

キャンバスは、ターミナル内で動作するビジュアルエディタです。ワークフロー図、セッションマップ、ノードグラフを作成でき、すべてASCII/Unicodeアートでレンダリングされます。

`p` を押してキャンバスパネルを切り替えます。

## キャンバスノード

ノードは作業の単位を表します。各ノードには以下の要素があります：

- **タイトル** — 短い説明
- **コンテンツ** — 詳細テキスト（複数行対応）
- **位置** — キャンバス上のx/y座標
- **コネクション** — 他のノードへのエッジ

### ノードの作成

1. `p` でキャンバスを開く
2. `n` を押して新しいノードを作成
3. ノードのタイトルを入力
4. 矢印キーで位置を調整
5. `Enter` を押して配置

### ノードの編集

ノードを選択して `e` を押すとコンテンツを編集できます。エディタは複数行テキストとワードラッピングをサポートしています。

## エッジ

ノード間をエッジで接続して、関係性やフローを表現します。

1. ソースノードを選択
2. `c` を押して接続を開始
3. ターゲットノードに移動
4. `Enter` を押してエッジを作成

### エッジラベル

エッジにはラベルを付けることができます。選択したエッジで `l` を押すとラベルを追加・編集できます。ラベルはエッジのパスに沿って表示されます。

## セッションマップ

キャンバスは**セッションマップ**を自動生成できます。これはアクティブなセッションとその関係を視覚的に表現したもので、以下のデータから構築されます：

- セッションライフサイクルイベント（開始、停止、再開）
- セッション間の関係データ
- AIが生成した要約

## ビューポートナビゲーション

| キー | 操作 |
| --- | --- |
| 矢印キー | ビューポートをパン |
| `+` / `-` | ズームイン/アウト（将来実装） |
| `h` | ビューポートをホームポジションにリセット |
| `f` | すべてのノードをビューに収める |

## AI統合（Max）

Maxティアでは以下が可能です：

- AIを使用してノード要約を生成
- セッション関係に基づいてノードを自動レイアウト
- タイプ別に色分けされた関係エッジを視覚化
